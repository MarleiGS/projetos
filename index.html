Aula: O que são Media Queries?

Neste capítulo falaremos sobre as Media Queries e nesta aula falaremos para que serve as Media Queries.

As Media Queries são extremamente importante nos dias de hoje, pois temos vários tipos de dispositivos com diferentes tamanho de telas, e.g. celulares, notebooks, TVs, monitores, tablets e outros.

Podemos consumir conteúdo em todos esses dispositivos ou podemos entrar em um site e imprimir o conteúdo desse site.

Tudo isso são mídias diferentes, ou seja, são lugares diferentes onde podemos exibir um conteúdo, portanto, necessitamos aprender Media Queries.

Media Queries na prática

Acesse os sites do YouTube e Globo.com.

⇒ Globo.com
✓ Na grande maioria dos sites, temos uma ou algumas versões de estilo para visualização em tela de diferentes tamanhos e outra versão para impressão.
⇒ YouTube
✓ No site do YouTube, podemos perceber que quando acessamos em um notebook temos uma versão com 4 vídeos e um menu lateral, mas ao diminuir o site ou trocar de tela, teremos uma outra versão de estilo para visualizar.
